# my-account  个人记账平台 


一、前端 my-account-web 
 1. vue2.0 + webpack + vue-cli + vue-router + vuex
 
二、后台 my-account-server  
 1. nodejs + express + mysql
 
三、设计需求  
 1. 登录:个人账户信息  
 2. 记录个人每日进出账务信息，主要字段包括：日期 类型 金额 成员 备注   
 3. 可根据日、月、年、类型筛选账务列表  
 4. 可通过柱状图、饼图、折线图展示账务情况  
 5. 首页：账户余额、本月支出、本月收入
